# Brain_Tumor_Detection_CNN
Developed an AI-powered brain tumor detection system using a hybrid approach combining Morphological Image Filtering and Convolutional Neural Networks (CNN)
Achieved 97.12% classification accuracy, outperforming standalone CNN baselines by 4.07%. The system preprocesses MRI images using morphological operations (erosion, dilation, opening, closing) before feeding them into a 3-layer deep CNN for feature extraction and Softmax classification.
Built with Python · TensorFlow/Keras · OpenCV · NumPy · Scikit-learn.
